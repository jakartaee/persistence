Guide to Jakarta EE Persistence TCK 3.2 Documentation

The Jakarta EE Persistence TCK 3.2 documentation includes the following:

  • The Jakarta EE Persistence TCK Release Notes provides the information on the test harness, the test suite and the platforms tested.
    • Updates to the Release Notes, if needed are available from the Jakarta EE Persistence Specification Web Site: https://jakarta.ee/specifications/persistence/3.2/
  • The Jakarta EE Persistence Technology Compatibility Kit, Version 3.2 User's Guide (or in PDF) provides the information that you need to install, set up, and run the Jakarta EE Persistence TCK, Version 3.1. In addition, the guide provides the rules you must comply with to pass the Jakarta EE Persistence TCK.
  • The Javadoc Assertion List lists all the javadoc assertions that are tested by the Jakarta EE Persistence TCK.
  • The Specification Assertion List lists all the specification assertions that are tested by the Jakarta EE Persistence TCK.
